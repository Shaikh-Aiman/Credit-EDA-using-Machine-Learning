- __os –__  Provides functions to interact with the operating system, such as file and directory handling.

- __warnings –__  Used to suppress or manage warning messages during program execution.

- __numpy –__  Supports high-performance numerical computations and array-based operations.

- __pandas –__  Enables efficient data loading, cleaning, and manipulation using DataFrames.

- __matplotlib –__  Used for creating static, interactive, and publication-quality visualizations.

- __seaborn –__  Built on Matplotlib, it provides high-level statistical data visualization with attractive default styles.

- __joblib –__  Facilitates efficient saving and loading of trained machine learning models.

- __scikit-learn (sklearn) –__  Provides machine learning algorithms, preprocessing tools, and evaluation metrics.

- __LogisticRegression –__  Implements a linear classification algorithm commonly used for binary classification problems.

- __RandomForestClassifier –__  An ensemble learning method that builds multiple decision trees to improve prediction accuracy.

- __Pipeline –__  Streamlines preprocessing and modeling steps into a single unified workflow.

- __StandardScaler –__  Standardizes features by removing the mean and scaling to unit variance.

- __train_test_split –__  Splits the dataset into training and testing subsets for model evaluation.

- __accuracy_score –__  Measures the proportion of correctly classified instances.

- __classification_report –__  Generates detailed performance metrics including precision, recall, and F1-score.

- __confusion_matrix –__  Displays the number of correct and incorrect predictions for each class.

- __roc_auc_score –__ Evaluates a model’s ability to distinguish between classes using the ROC curve.

- __TensorFlow (Keras) –__  A deep learning framework used to build and train neural network models.

- __Sequential –__  Allows construction of neural networks layer by layer in a linear sequence.

- __Dense –__  Implements a fully connected neural network layer.

- __Dropout –__  Reduces overfitting by randomly disabling neurons during training.

- __EarlyStopping –__ Stops training when model performance stops improving, preventing overfitting.
